# AR Tracing PWA

Upload any image and overlay it with adjustable opacity onto your phone's rear camera feed. Position your phone over a blank sheet of paper and trace the image directly.

## Features

- Rear camera live feed as tracing canvas
- Upload any image and adjust opacity via slider
- Drag, pinch-to-zoom, and two-finger rotation gestures
- Flip/mirror the image horizontally
- Lock controls to prevent accidental image movement
- Screen Wake Lock — keeps screen on while tracing
- Fully offline-capable PWA (add to home screen)

## Usage

1. Open **https://ivan-lyutskanov.github.io/ar-tracing-pwa/** on your phone
2. Allow camera access
3. Tap **Upload Image** to select a reference image
4. Adjust opacity with the slider
5. Position the image over paper using touch gestures
6. Start tracing

## Deploy

Serve from any static host or use GitHub Pages:

```
git clone https://github.com/ivan-lyutskanov/ar-tracing-pwa.git
```

No build step required — open `index.html` directly or serve with any HTTP server.
